CO2 - Coroutine The Second
===

A C++ stackless coroutine emulation library, providing interface close to [N4286](http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2014/n4286.pdf).

## Dependencies

- [Boost](http://www.boost.org/)
